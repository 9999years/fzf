# Fzf

**fzf** provides efficient keybindings for the [fzf](https://github.com/junegunn/fzf) utility.

## Install

With [Fisherman]

```
fisher install jethrokuan/fzf
```

## Usage
See [fzf](https://github.com/junegunn/fzf)
